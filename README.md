### Como usar?

#### Instale o Git:

    sudo apt install git

#### Faça um git clone:

    git clone https://github.com/JuanBindez/pyREC-v1.0.0
    
#### Acesse a pasta:

    cd pyREC-v1.0.0/
    
#### Agora é só rodar o software:

    python3 pyrec.py
